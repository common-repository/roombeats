Roombeats Plugin allows you to add Roombeats overlays to your images.

== Description ==

If you have your Roombeats account (http://roombeats.com) then install
this plugin to easily enhance your images with all Roombeats goodies.

You can grab your Roombeats Website ID or the full script from your
dashboard (http://roombeats.com/dashboard) and configure this plugin.
